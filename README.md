# Supply Chain Analytics using PySpark

## Project Overview
This project demonstrates a real-time PySpark ETL pipeline for supply chain analytics.

## Features
- Data ingestion
- Data cleaning
- Audit columns
- Profit calculation
- Shipping delay calculation
- Window functions
- Aggregations
- Parquet output

## Tech Stack

- Python
- Spark SQL
- GitHub
